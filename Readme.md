# B350m-bazooka Hackintosh

## 硬件配置

| 系统     | Mac OS Ventura          |
| -------- | ----------------------- |
| 主板     | msi b350m bazooka       |
| 处理器   | r5 2600 (6c 12t)        |
| 内存     | 毕伟4g 2400mhz ddr4  x4 |
| 散热     | 玄冰400                 |
| 硬盘     | 光威固态256g            |
| 有线网卡 | RealtekRTL8111          |
| 无线网卡 | BCM94331                |

OpenCore版本 v0.9.5

支持的系统版本：Mac OS Ventura（13）

更新日期 2023.10.11



## 正常工作的功能

1. 有线网卡正常
2. 睡眠唤醒正常
3. 前后端声音输出
4. 所有USB端口

## 未测试的功能

1. 内置麦克风
2. FaceTime

## 注意事项

1. 需要改**<u>三码</u>**
2. 若iCloud无法登陆，需重新内建网卡（已用RealtekRTL8111内建）
